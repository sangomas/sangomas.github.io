xampp for lamp & win

[using grid.css]

Linux:


$ sudo -s
$ apt update
$ apt upgrade

downloading Bitnami xampp:
$ wget https://www.apachefriends.org/xampp-files/7.1.10/xampp-linux-x64-7.1.10-0-installer.run

execute and install:
$ chmod +x xampp-linux-x64-7.1.10-0-installer.run
$ ./xampp-linux-x64-7.1.10-0-installer.run

start all xampp services:
$ /opt/lampp/xampp start

apache only:
$ /opt/lampp/xampp startapache

proftpd server only:
$ /opt/lampp/xampp startftp

mysql database server only:
$ /opt/lampp/xampp startmysql

stop all sevices:
 $ /opt/lampp/xampp stop

stop apache only:
$ /opt/lampp/xampp stopapache

stop proftpd FTP server:
$ /opt/lampp/xampp stopftp

stop mysql database server:
$ /opt/lampp/xampp stopmysql

restart:
$ /opt/lampp/xampp restart

help:
/opt/lampp/xampp --help

GUI install:
error: Error importing pygtk2 and pygtk2-libglade:
sudo apt-get install python-glade2
sudo apt-get install python-gtk2
start GUI:
gksu "python /opt/lampp/share/xampp-control-panel/xampp-control-panel.py"
